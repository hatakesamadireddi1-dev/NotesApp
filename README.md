# 📝 Notes App

A modern, responsive Notes App built using React and Vite, supporting full CRUD operations and a smooth user experience.
This project was recently rebuilt and redeployed using AI-assisted development to improve speed, debugging, and overall code quality.

---

## 🚀 Live Demo

👉 [https://notesapp-mauve-three.vercel.app/](https://notesapp-mauve-three.vercel.app/)

---

## ✨ Features

* 📝 Create, edit, and delete notes (CRUD)
* 🔍 Search and organize notes
* 📱 Fully responsive design
* ⚡ Fast and optimized performance
* 🔗 API integration support

---

## 🛠️ Tech Stack

* **Frontend:** React.js, JavaScript, HTML, CSS
* **Build Tool:** Vite
* **State Management:** React Hooks
* **Version Control:** Git, GitHub
* **Deployment:** Vercel
